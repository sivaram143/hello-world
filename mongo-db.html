<h1 id="mongodb">MongoDB</h1> <h3 id="introduction">Introduction</h3> <ul> <li>MongoDB is an open source document oriented database.</li> <li>It is fall under NoSQL (Not Only SQL)<ul> <li>Document databases (Eg: MongoDB)</li> <li>Graph stores (Eg: Neo4J)</li> <li>Key-value stores (Eg: Redis)</li> <li>Wide-column stores (Eg: Cassandra, HBase)</li> </ul> </li> <li>It provides high performance, high availability, and automatic scaling.</li> </ul> <h4 id="terminology">Terminology</h4> <table> <thead> <tr> <th>RDBMS</th> <th>MongoDB</th> </tr> </thead> <tbody> <tr> <td>Database</td> <td>Database</td> </tr> <tr> <td>Table</td> <td>Collection</td> </tr> <tr> <td>Row</td> <td>Document</td> </tr> <tr> <td>Column</td> <td>Field</td> </tr> <tr> <td>table joins</td> <td>embedded documents and linking</td> </tr> </tbody> </table> <h3 id="setup">Setup</h3> <p>I. Importing the Public Key</p> <pre><code> sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv 7F0CEB10 </code></pre><p>II. Create source list file MongoDB</p> <pre><code> echo &quot;deb http://repo.mongodb.org/apt/ubuntu &quot;$(lsb_release -sc)&quot;/mongodb-org/3.0 multiverse&quot; | sudo tee /etc/apt/sources.list.d/mongodb-org-3.0.list </code></pre><p>III. Update the repository</p> <pre><code> sudo apt-get update </code></pre><p>IV. Install MongoDB</p> <pre><code> sudo apt-get install -y mongodb-org (latest stable) sudo apt-get install -y mongodb-org=3.0.12 mongodb-org-server=3.0.12 mongodb-org-shell=3.0.12 mongodb-org-mongos=3.0.12 mongodb-org-tools=3.0.12 (specific release) mongod -version (Check version) </code></pre><p>V. Start/Stop/Restart MongoDB </p> <pre><code> sudo service mongod start/stop/restart </code></pre><p>Note: <strong><em> Some Configuration Files </em></strong></p> <ul> <li>Logs: /var/log/mongodb/mongod.log</li> <li>Configuration : /etc/mongod.conf</li> </ul> <h3 id="connecting-to-database">Connecting to Database</h3> <ul> <li>starting mongo server <code>mongod</code></li> <li>starting mongo Client <code>mongo</code> (JS based shell)</li> <li>Remote Database:</li> <li>mongos: Sharding Cluster</li> <li>Basic Shell Commands</li> </ul> <table> <thead> <tr> <th>Command</th> <th>Description</th> </tr> </thead> <tbody> <tr> <td>help</td> <td>Show help</td> </tr> <tr> <td>db</td> <td>Show Current Database</td> </tr> <tr> <td>db.help()</td> <td>Show help for database methods</td> </tr> <tr> <td>show users</td> <td>Print a list of users for current database.</td> </tr> <tr> <td>show roles</td> <td>Print a list of all roles</td> </tr> <tr> <td>show dbs</td> <td>Print a list of all databases on the server</td> </tr> <tr> <td>use <db></td> <td>Switch current database to <db>.</td> </tr> <tr> <td>show collections</td> <td>Print a list of all collections for current database</td> </tr> <tr> <td>load()</td> <td>Execute a JavaScript file</td> </tr> <tr> <td>db.stats()</td> <td>To show the stats</td> </tr> <tr> <td>db.addUser</td> <td>To add the User</td> </tr> <tr> <td>runProgram(&quot;ls&quot;, &quot;-l&quot;)</td> <td>To Run OS Commands</td> </tr> <tr> <td>db.runCommand({cursorInfo:1})</td> <td>Cursor Info</td> </tr> <tr> <td>db.shutdownServer()</td> <td>Shutdown the process</td> </tr> </tbody> </table> <ul> <li>Running script <code>mongo &lt;script&gt;.js</code></li> <li>Object ID : A 12-byte Hexa decimal BSON type</li> </ul> <h3 id="crud-operations">CRUD Operations</h3> <p><strong>C</strong> - CREATE | <strong>R</strong> - READ/RETRIEVE | <strong>U</strong> - Update | <strong>D</strong> - DELETE</p> <p>A. Create a Database</p> <pre><code> &gt; use &lt;database&gt; </code></pre><p>B. Create a Collection</p> <pre><code> &gt; db.collection.insert({field:value,...}); </code></pre><p>C. Insert a document</p> <pre><code> &gt; db.collection.insert({field:value,...}); </code></pre><p>D. Retrieve all documents </p> <pre><code> &gt; db.collection.find() </code></pre><p>E. Retrieve specified fields in documents </p> <pre><code> &gt; db.collection.find({}, {field1:true,field2:true}) </code></pre><p>F. Query for One document</p> <pre><code> &gt; db.collection.findOne() </code></pre><p>G. Update a document</p> <pre><code> &gt; db.collection.update({_id:value},{$set:{field:value}},{multi:true}) </code></pre><p>H. Delete all documents</p> <pre><code> &gt; db.collection.remove() </code></pre><p>I. Deleting specific document </p> <pre><code> &gt; db.collection.remove({field:value}) </code></pre><p>J. Delete Database</p> <pre><code> &gt; use database &gt; db.dropDatabase() </code></pre><h3 id="misc">Misc</h3> <ul> <li><p>Import</p> <pre><code>$ mongoimport --db database --collection collection &lt; path/filename.json </code></pre></li> <li><p>Export</p> <pre><code>service mongod stop (To Stop before backup) $ mongoexport --db database --collection collection --out path/ filename.json $ mongodump --dbpath /data/db --out path </code></pre></li> <li><p>Monitoring</p> <pre><code>$ mongostat // captures database operations by type $ mongotop // tracks read/write activity $ mongoperf // performance testing tool MMS ( MongoDB Monitoring Service) </code></pre></li> <li><a href="https://blog.ajduke.in/2013/05/31/setup-mongodb-replica-set-in-4-steps/">Replica</a></li> <li><a href="https://www.javacodegeeks.com/2015/02/setting-up-sharded-mongodb-cluster-in-localhost.html">Sharding</a></li> <li><a href="https://docs.mongodb.com/manual/core/map-reduce-concurrency/">MapReduce</a></li> </ul> <h3 id="references">References</h3> <ul> <li><a href="docs.mongodb.org">Mongo</a></li> <li><a href="http://mongodb.github.io/mongo-java-driver/3.0/driver/getting-started/quick-tour/">MongoDB Driver</a></li> <li><a href="http://3t.io/mongochef/">MongoChef</a></li> <li><a href="https://github.com/mongodb">Mongo-github</a></li> </ul>
