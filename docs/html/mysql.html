Create a Database</p> <pre><code> mysql&gt; CREATE DATABASE database; </code></pre><p>B. Create a Table</p> <pre><code> mysql&gt; CREATE TABLE table(column datatype,...); </code></pre><p>C. Insert a row</p> <pre><code> mysql&gt; INSERT INTO table(column,...) VALUES(value,...); </code></pre><p>D. Retrieve all rows</p> <pre><code> mysql&gt; SELECT * FROM table; </code></pre><p>E. Retrieve specified fields in documents </p> <pre><code> mysql&gt; SELECT column1,column2 FROM table; </code></pre><p>F. Retrieve specific rows</p> <pre><code> mysql&gt; SELECT * FROM table WHERE column = value; </code></pre><p>G. Update a row</p> <pre><code> mysql&gt; UPDATE table SET column = value WHERE id = id; </code></pre><p>H. Delete all rows</p> <pre><code> mysql&gt; DELETE FROM table; </code></pre><p>I. Deleting specific document </p> <pre><code> mysql&gt; DELETE FROM table WHERE column = value; </code></pre><p>J. Delete Database</p> <pre><code> mysql&gt; DROP DATABASE database; </code></pre><h3 id="misc">Misc</h3> <ul> <li><p>Import</p> <pre><code>mysql&gt; LOAD DATA LOCAL INFILE &#39;dump.txt&#39; INTO TABLE tbl; (or) $ mysqlimport -u root -p --local database_name dump.txt $ mysql -u [username] -p -h localhost [database] &lt; db_backup.sql </code></pre></li> <li>Export<pre><code>mysql&gt; SELECT * FROM tbl INTO OUTFILE &#39;/tmp/filename.txt&#39;; (or) $ mysqldump -u root --opt database &gt; path.filename.sql </code></pre></li> <li><a href="https://www.nagios.com/solutions/mysql-monitoring/">Monitoring</a></li> <li><a href="https://www.phpmyadmin.net/">PHPMYADMIN</a></li> <li><a href="https://dev.mysql.com/downloads/workbench/">MySql Workbench</a></li> <li><a href="http://www.tecmint.com/mysqladmin-commands-for-database-administration-in-linux/">MySql Admin Commands</a></li> </ul> <h3 id="references">References</h3> <ul> <li><a href="http://dev.mysql.com/doc/">MySQL</a></li> <li><a href="http://www.sqlteaching.com/">http://www.sqlteaching.com/</a></li> <li><a href="https://www.youtube.com/playlist?list=PLfdtiltiRHWEw4-kRrh1ZZy_3OcQxTn7P">https://www.youtube.com/playlist?list=PLfdtiltiRHWEw4-kRrh1ZZy_3OcQxTn7P</a></li> </ul>
