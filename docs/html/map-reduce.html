**[Back to the main page](index.html)** <h3 id="map-reduce">Map Reduce</h3> <ul> <li><p>Map-reduce is a data processing paradigm for condensing large volumes of data into useful aggregated results. For map-reduce operations, MongoDB provides the mapReduce database command.</p> </li> <li><p>The mapReduce command takes 2 primary inputs:</p> <ul> <li>the mapper function</li> <li>the reducer function.</li> </ul> </li> <li><p>A Mapper will start off by reading a collection of data and building a Map with only the required fields we wish to process and group them into one array based on the key. And then this key value pair is fed into a Reducer, which will process the values.</p> </li> </ul> <h3 id="references-">References:</h3> <ul> <li><a href="https://github.com/sedouard/mongodb-mva/tree/master/module4_advanced_data_ops">https://github.com/sedouard/mongodb-mva/tree/master/module4_advanced_data_ops</a></li> <li><a href="http://thejackalofjavascript.com/mapreduce-in-mongodb/">http://thejackalofjavascript.com/mapreduce-in-mongodb/</a></li> </ul>
